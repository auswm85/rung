## [0.1.0] - 2026-01-23

### Added

- Initial release
- Stack visualization in sidebar with branch status indicators
- Branch checkout by clicking in tree view
- Sync command to rebase branches onto updated parents
- Continue/abort sync for conflict resolution
- Submit command to push and create/update PRs
- Compare command with VS Code diff editor integration
- Open PR in browser
- Navigate up/down the stack
- Create branch with name or commit message
- Doctor diagnostics with structured output
- Undo last sync operation
- Merge PR and cleanup branch
- Initialize rung in repository
- Status bar showing current branch and stack position
- Auto-refresh on file save and git events
- Configurable CLI path and refresh settings
